# Frontend Specification

## Technology Stack
- Framework: Next.js (React)
- Styling: Tailwind CSS / Custom Dark Theme
- Icons: Lucide / Radix Icons

## Core Layouts
1. **Sidebar Navigation**: Left-side vertical navigation for switching between Agent Templates, UI Components, and Magic Chat.
2. **Main Display Stage**: 
   - Feature spotlight with horizontal scrolling for component sections.
   - Integrated live code-preview split-pane for Magic Chat.

## Interactions & Aesthetics
- **Dark Mode Default**: Premium enterprise aesthetic with high contrast and subtle glowing accents.
- **Micro-animations**: Hover states and transition animations utilizing Framer Motion or GSAP.
- **Live Previewing**: Component cards instantly render the code without requiring a full page refresh.
