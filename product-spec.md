# 21st Builder | AI Component Marketplace - Product Specification

## Overview
A high-end, dark-themed AI component marketplace and agent foundry inspired by 21st.dev. It provides developers and designers with open-source UI components and specialized AI Agents via the 21st Agents SDK.

## Target Audience
- Frontend and Full-stack Developers
- UI/UX Designers
- AI Engineers shipping agentic products

## Core Features
1. **Agent Templates & SDK Sandbox**: Starter kits (Next.js, Python, Go) for quickly building and deploying AI agents (Lead Research, Support, Code Analysis).
2. **Component Marketplace**: Horizontally scrollable galleries of meticulously designed, copy-pasteable UI components.
3. **Magic Chat**: An integrated chat interface for generating custom components on the fly (1Code), eliminating manual boilerplate.
4. **Agent Observability**: Built-in chat history, spend limits, and sandbox execution monitoring.
